# MindDo AI · 全球上市宣传计划 (Global Go-to-Market Plan)

面向**全球市场、以美国学生为主**的「Every Kid's AI Learning Companion」**MindDo AI**
新品上市的完整宣传作战计划，以市场负责人视角制定，单文件网页形式输出，方便分享给团队与管理层。
内部规划用中文，对外文案以英文呈现。

## 📄 内容

打开 [`index.html`](index.html) 查看，覆盖：

| # | 模块 | 关键产出 |
|---|------|---------|
| 01 | 概览 | 一页看懂战略目标、核心打法、北极星指标 |
| 02 | 目标人群 | 焦虑妈妈 / 科技爸爸 / 孩子 三类画像 |
| 03 | 信息架构 | 品牌主张 + 三大支柱 + 分人群 Hook/CTA |
| 04 | 时间线 | 12 周三阶段（筹备→预热→引爆→增长）含 4 个 Go/No-Go 门槛，对齐美国开学季 |
| 05 | 渠道矩阵 | Instagram/TikTok/YouTube/Email+社区/Creators 等渠道角色与优先级 |
| 06 | 内容日历 | 分阶段素材组合与「内容铁律」 |
| 07 | 准备清单 | 8 大模块可勾选 readiness checklist（进度存本地浏览器） |
| 08 | 预算 | 分配比例框架 + 环形图 |
| 09 | KPI | 转化漏斗 + 核心指标 + 复盘节奏 |
| 10 | 合规与风险 | COPPA / FERPA / GDPR-K / FTC 红线 + 发布前合规闸门 |
| 11 | 团队分工 | RACI 责任矩阵 + 关键会议节奏 |

## ✨ 功能

- 单文件、零构建：HTML + 内联 CSS + Chart.js（CDN）
- 可勾选的准备清单，进度自动保存在本地浏览器
- 滚动高亮导航（scrollspy）、响应式、支持打印导出 PDF

## 🚀 发布到 GitHub Pages

1. 推送本仓库到 GitHub。
2. 打开仓库 **Settings → Pages**。
3. **Source** 选择 **Deploy from a branch**，**Branch** 选 `main` / 根目录 `/`，保存。
4. 约 1 分钟后访问 `https://<用户名>.github.io/MindDo_Marketing/`。

## 👀 本地预览

直接用浏览器打开 `index.html`；或：

```bash
python -m http.server 8000
# 访问 http://localhost:8000
```

> ⚠️ 计划中的转化率、CAC、漏斗比例等为**示意性框架**，须以真实试点数据校准；合规相关（COPPA/FERPA/GDPR/FTC）以美国本地法务的最新意见为准。
